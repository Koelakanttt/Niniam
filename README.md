## Niniam Modeler
https://niniam-modeler.online
Веб-редактор диаграмм с поддержкой нотаций BPMN, UML и IDEF0.

Стек: React 19, TypeScript, ReactFlow, Zustand, Tailwind CSS / Node.js, Express, Prisma, PostgreSQL, WebSocket, Docker.

Ключевые возможности: визуальное моделирование процессов, совместная работа в реальном времени, AI-ассистент для генерации диаграмм по текстовому описанию, симуляция BPMN-процессов, версионирование, мультипровайдерная AI-архитектура (Gemini, Groq, YandexGPT).
