# 🏛️ Niniam Modeler

**[→ Открыть веб-редактор](https://niniam-modeler.online)**

![Веб-редактор диаграмм](https://github.com/user-attachments/assets/03c90e68-bb99-4ef1-b8cb-9b50b34a93c4)

</div>

Веб-редактор диаграмм с поддержкой нотаций **BPMN**, **UML** и **IDEF0**.  
Совместная работа в реальном времени, AI-ассистент, симуляция процессов и версионирование — всё в одном инструменте.

---

## 🧩 Стек технологий

**Frontend**  
`React 19` · `TypeScript` · `ReactFlow` · `Zustand` · `Tailwind CSS`

**Backend & Infra**  
`Node.js` · `Express` · `Prisma` · `PostgreSQL` · `WebSocket` · `Docker`

---

## ⚡ Ключевые возможности

- 🎨 **Визуальное моделирование** — BPMN, UML, IDEF0 на интуитивном холсте
- 👥 **Совместная работа** — редактируйте диаграммы с командой в реальном времени
- 🤖 **AI-ассистент** — генерируйте диаграммы по текстовому описанию (Gemini, Groq, YandexGPT)
- ▶️ **Симуляция BPMN** — проверяйте логику процессов до внедрения
- 🔄 **Версионирование** — коммиты и возврат к любой версии
- 📥 **Импорт моделей** — загружайте диаграммы из внешних систем
- 💻 **Генерация кода** — автоматически создавайте код по диаграмме
